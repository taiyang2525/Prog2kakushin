# Prog2kakushin
## 概要
このレポジトリは2023年度広島市立大学情報科学部「プログラミングII」での自己制作課題プログラムを公開している（Python3, Jupiter Notebook形式）
コメントなどいただければ幸いだ．

### Work1
#### 説明
Work1はユーザにそれぞれの都道府県名に対応する数字（0から46の整数）を求めることで，気象情報を出力するプログラムである．
